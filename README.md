# Boboter

An ESP-IDF based firmware for the robot built in the HTL St. PÃ¶lten's EL department.

Documentation is provided under `docs`.
When using CLion, the project comes with configuration files for building, uploading and generating doxygen.

### Commands
To build doxygen documentation, run `doxygen Doxyfile` in the project's root directory.
To build the project, use `pio run` and to upload it to the microcontroller, use `pio run -t upload`.
